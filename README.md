# 📚 Sobre o Repositório

Este repositório reúne exercícios desenvolvidos durante os estudos de programação e sistemas operacionais.

O objetivo é praticar desde os fundamentos da lógica até conceitos mais próximos do ambiente real de desenvolvimento, utilizando Python e ferramentas relacionadas ao ecossistema de sistemas operacionais.

Conceitos estudados e desenvolvidos:

- Lógica de programação
- Vetores e matrizes
- Threads e multiprocessing
- Manipulação de arquivos
- Docker
- Máquina virtual Linux
- Conceitos de sistemas operacionais
